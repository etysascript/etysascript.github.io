Introduction

Welcome to the Etysa reference documentation! This is intended as a resource for people who already have some familiarity with Etysa and want to learn more about using it.

If that's not you (yet), you may prefer to visit the interactive tutorial or the examples before consulting this reference. You can try Etysa online using the REPL. Alternatively, if you'd like a more fully-featured environment, you can try Etysa on StackBlitz.
